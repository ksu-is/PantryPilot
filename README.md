# PantryPilot
### Intelligent Zero-Waste Kitchen Assistant

PantryPilot is a Python-based application designed to help users reduce food waste and save money. By tracking a user's current pantry inventory, the app suggests recipes that maximize existing ingredients and prioritze items nearing their expiration date.

## Features
- **Virtual Pantry** Digital inventory management.
- **Smart Recipe Search:** Powered by the Spoonacular API.
- **Substitution Engine:** Suggestions for missing non-essential ingredients.

## Tech Stack
- Python 3.x
- SQLite3 (Database)
- Spoonacular API
